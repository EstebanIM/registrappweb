Registrapp

Registrapp es una aplicación creada para poder registrar la asistencia de los alumnos mediante un codigo QR.

Descargar Aplicación

Para instalar nuestra aplicación tienes que habilitar esta opción para activar el modo desarrollador:

Android 9 y versiones posteriores:

Configuración > Acerca del dispositivo > Presionar 7 veces el número de compilación.

Android 8.0.0 y Android 8.1.0:

Configuración > Sistema > Acerca del dispositivo > Presionar 7 veces el número de compilación.

Despues de realizar los pasos podrás instalar nuestra aplicación
